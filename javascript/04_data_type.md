# Data Type

## 🔖 Dynamic Typing

```js
let changable = 'beanie'; // 문자열 할당

changable = 10; // 숫자형으로 변경

changable = true; // boolean형으로 변경
```
- dynamic typing : 동적 타이핑, type이 바뀔 수 있음
- 변수가 가지는 고정 타입이 없다.
- 타입이 없는 것은 아니다.
